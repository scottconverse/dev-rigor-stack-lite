# Dev Rigor Stack — Audit Lite

Read `../quick-audit-lite/SKILL.md` completely and apply it without abbreviation. That
sibling is the canonical implementation and short-name `$quick-audit-lite` entrypoint.

Consume PLAN, BUILD, and Proof Gate evidence when present. Escalate user-facing runtime
work to `$dev-rigor-stack-lite-walkthrough`, public surfaces to
`$dev-rigor-stack-lite-visitor-audit`, and high-blast changes to
`$dev-rigor-stack-lite-audit-team`. A light lane may narrow scope; it may not weaken the checks
that apply inside that scope.
